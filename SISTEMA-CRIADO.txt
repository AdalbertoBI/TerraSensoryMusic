🚀 COMO USAR O SISTEMA:

1. INSTALAÇÃO (Apenas uma vez):
   execute: install.bat

2. INICIALIZAÇÃO (A cada uso):
   execute: start.bat
   OU diretamente: node server-demo.js

3. ACESSO:
   Abra o navegador em: http://localhost:3000

4. MONITORAMENTO:
   • Clique em "Conectar" na interface
   • Observe dados MIDI em tempo real
   • Use filtros e controles conforme necessário
   • Explore as abas de análise avançada

🔧 COMPATIBILIDADE TERRA ELETRÔNICA:

✅ Hardware Suportado:
   • Receptor MIDI 32U4 (USB)
   • Sistema RF24 (wireless)  
   • Slaves 01-05 (configuráveis)

✅ Protocolo MIDI:
   • Canais 0x90-0x9F (16 canais)
   • Baud rate 31250 (padrão MIDI)
   • RF24 channels: 60, 70, 80, 90

✅ Tipos de Mensagem:
   • Note On/Off (notas musicais)
   • Control Change (modulação)
   • Pitch Bend (alteração de altura)
   • System (clock e sincronização)

🛠️ MODO DE DESENVOLVIMENTO:

Para conectar dispositivos MIDI reais:
1. Instale biblioteca MIDI nativa:
   npm install midi
   
2. Use o servidor completo:
   node server.js

3. O sistema detectará automaticamente
   dispositivos MIDI-Terra conectados
